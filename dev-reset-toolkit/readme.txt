=== Dev Reset Toolkit ===
Contributors: devreset
Requires at least: 6.0
Tested up to: 6.7
Requires PHP: 7.4
Stable tag: 2.0.0
License: GPLv2 or later

Developer-focused reset utility with safe reset types, tools, snapshots, reactivation, logs, and settings.

== Installation ==
1. Zip the `dev-reset-toolkit` folder.
2. Upload from Plugins > Add New > Upload Plugin.
3. Activate Dev Reset Toolkit.
